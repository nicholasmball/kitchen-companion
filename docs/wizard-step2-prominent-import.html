Prominent Import Button

Import elevated to a card with visual weight. Custom item demoted to a text link.

Rationale
Hierarchy change:
1. Recipe grid — primary action, most common path
2. Import card — elevated to a styled card with primary border, icon, and arrow. Same visual language as the recipe grid but distinct. Clearly says what it does.
3. Custom item — demoted to a collapsed text link ("+ Add a custom item without a recipe"). Clicking expands the mini-form. Out of the way, but accessible if needed.

Why a card: The import button now has the same visual weight as the recipe cards. It looks like a first-class option, not an afterthought. The primary-coloured border and icon make it stand out without being obnoxious. The arrow hints that clicking opens a dialog.

Why demote custom items: If it won't be used, it shouldn't take up prime real estate. A text link is enough — power users who need it will find it, and it doesn't distract everyone else.
